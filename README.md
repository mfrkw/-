# Matchingbow

## サイト概要

犬を触りたい・飼いたい人と遊んでもらいたい保護犬をマッチングさせるサイトです。

### サイトテーマ

犬と人が気軽に遊べることをテーマにしました。

### テーマを選んだ理由

犬を飼う人が増えてきていますが、飼う前にマッチングして遊んでみることで犬を飼うという責任や大変さを少しでも分かって貰えればと思いました。保護犬と人をつなげて最終的に保護犬の殺処分数を減らしていければと考えました。


### ターゲットユーザ

- 犬を飼いたいけど飼えない人 
- 犬を飼うことを検討しているファミリー層（30代）

### 主な利用シーン

- 犬をモフモフしたいとき
- 犬を散歩させたいとき
- 犬に癒やされたいとき
